# Ts3Proxy

Ts3Proxy is a simple and useful JavaScript Node.js framework for Teamspeak IP forwarding (Teamspeak Proxy).

## Features
- **Lightweight** and easy to use.
- No dependencies required.
- Straightforward setup.

## Getting Started
### Running the Proxy
To start the Teamspeak Proxy, simply run one of the following commands in your terminal:

```bash
node .
```
or:
```bash
node server.js
```

## Installation Notes
No need to download or install any dependencies. You can directly run the proxy without executing `npm install`.
